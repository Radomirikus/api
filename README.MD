# Праздничные дни
## получение токена
Зарегистрируйтесь на сайте `https://calendarific.com/api-documentation`, в личном кабинете найдите `API Key` и скопируйте.
## добавления токена в код
В проекте создайте файл `.env` и напишите туда
```
TOKEN="Вставьте ваш токен который вы скопировали ранее"
```
Запустите проект.
